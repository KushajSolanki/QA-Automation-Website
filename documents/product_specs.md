# Product Specifications

### Pricing Rules
- Product A costs $100.
- Product B costs $50.

### Discount Rules
- The discount code **SAVE15** applies a 15% discount on the total.
- The discount code must be case-insensitive.
- If an invalid discount code is entered, the system must show an error message.
- The error message must NOT change the total price.

### Shipping Rules
- Standard shipping is **free**.
- Express shipping adds an additional **$10** to the final amount.

### Payment Rules
- User must select a payment method before clicking Pay Now.
- Supported payment methods: **Credit Card**, **PayPal**.

### Order Success
- If all fields are valid, clicking Pay Now must display: **"Payment Successful!"**
